Create a New Projectâ€‹

  1. Open Projects app
  2. Click "+ Add"
  3. Fill required fields: Job #, Job Name, Division, PM, OC
  4. Save â†’ Stages auto-create in ~2 min

Advance to Next Stageâ€‹

  1. Open project
  2. Change "Current Stage Status" to "[Stage] Active"
  3. Wait 2-3 min for tasks to appear

Complete a Taskâ€‹

  1. Open task
  2. Change Status to "Complete"
  3. Dependent tasks auto-advance (if any)

Add a Contactâ€‹

  1. Open Contacts app
  2. Click "+ Add"
  3. Fill: Name, Job Title, Organization, Email, Phone
  4. Save

Link Contact to Projectâ€‹

  1. Open project
  2. Find "Customer / GC" field
  3. Start typing contact name, select from list
  4. Save

Create an RFIâ€‹

  1. Add task in Pre-Con or Construction stage
  2. Name: "RFI: [Description]"
  3. Status: In Progress
  4. Document question + response in Notes
  5. Attach supporting docs

Create a Change Order Requestâ€‹

  1. Add task in appropriate stage
  2. Name: "COR: [Description]"
  3. Status: In Progress
  4. Document: what changed, why, cost/time impact
  5. Attach: photos, emails, cost breakdown
