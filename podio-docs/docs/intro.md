# Welcome to PSS Podio Documentation

This documentation will help you learn how to use Podio at Precision Site Services.

## What is Podio?

Podio is our project management system that helps us track:
- Projects from bid to completion
- Tasks and their dependencies
- Team assignments and responsibilities
- Documents and files
- Customer and vendor contacts
